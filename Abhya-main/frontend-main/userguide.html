The Home Page

This is the main page of the website. It shows different camera views and gives an alert if any violence is detected. It also has an option to overview the application and guide on how to use it.

Investigation Page

This is the page which pops up if the person clicks on the video which gives the alert. The person will receive the location and video and he can choose whether to respond or ignore. The emergency button calls all the emergency services together.

The Action Page

This page opens up if the person clicks the Respond button. He/she will get an option to go and take the action himself or call any of the emergency services like ambulance, forensics,etc.

Overview Page

This page gives an overview of the product.
